1 PC and 1 Raspberry Pi join same network

### PC
Set hostname as karaage

```sh
$ git clone https://github.com/karaage0703/hendigi_camera
$ cd hendigi_camera/server
```

```python
$ python hendigi_server.py
```

### hendigi camera

```sh
$ python hendigi_camera.py
```
